# 🎨 WEB UI/UX GUIDELINES — Cortex

> This English page mirrors the Vietnamese source of truth.
> Please refer to the [Vietnamese version](../05_web_ui_ux_guidelines.md) for the complete document.

**Summary:** Search-first workspace. Command palette (`Cmd/Ctrl+K`) with Search vs Ask AI modes; AI answers always show citations. Core screens: Home/Search, Knowledge Detail, Wiki Editor (OCC conflict banner + diff), Question Detail, AI Assistant Chat, Credit Wallet, Org Admin, Reputation. Must surface distributed states: AI processing/unavailable (Circuit OPEN, no charge), out-of-credit, rate-limited (429), OCC conflict (409), indexing-pending. Trust UX: citations, Verified/Stale badges, explicit tenant context.
